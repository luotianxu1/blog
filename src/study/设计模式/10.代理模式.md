- 代理模式(Proxy),为其他对象提供一种代理以控制对这个对象的访问
- 代理模式使得代理兑现控制具体对象的引用。代理几乎可以是任何对象:文件、资源、内存中的对象，或者是一些难以复制的东西

```js
class Star {
    play() {
        console.log('演戏')
    }
}

class StarPaocy {
    constructor() {
        this.superStar = new Star()
    }

    talk(price) {
        if (price >= 10000) {
            this.superStar.play()
        } else {
            console.log('价钱不合适')
        }
    }
}

let jr = new StarPaocy()
jr.talk(10000)
```
